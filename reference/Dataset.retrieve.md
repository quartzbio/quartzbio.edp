# Dataset.retrieve

Retrieves the metadata about a specific dataset from QuartzBio EDP.

## Usage

``` r
Dataset.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Dataset.retrieve("1234567890")
} # }
```
