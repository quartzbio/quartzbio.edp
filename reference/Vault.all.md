# Vault.all

Retrieves the metadata about all accessible vaults.

## Usage

``` r
Vault.all(..., env = get_connection())
```

## Arguments

- ...:

  (optional) Additional query parameters (e.g. limit, offset).

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Vault.all()
} # }
```
