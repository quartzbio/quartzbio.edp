# Beacon.delete

Delete a specific beacon from QuartzBio EDP.

## Usage

``` r
Beacon.delete(id, env = get_connection())
```

## Arguments

- id:

  The ID of the beacon.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Beacon.delete("1234567890")
} # }
```
