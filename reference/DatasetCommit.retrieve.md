# DatasetCommit.retrieve

Retrieves the metadata about a specific dataset commit on QuartzBio EDP.

## Usage

``` r
DatasetCommit.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset commit.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetCommit.retrieve(12345)
} # }
```
