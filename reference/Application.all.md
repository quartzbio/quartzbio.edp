# Retrieve the metadata about all application on QuartzBio EDP available to the current user.

Retrieve the metadata about all application on QuartzBio EDP available
to the current user.

## Usage

``` r
Application.all(env = get_connection(), ...)
```

## Arguments

- env:

  Custom client environment.

- ...:

  Additional query parameters, passed to .request().

  <https://docs.solvebio.com/>

## Examples

``` r
if (FALSE) { # \dontrun{
Application.all()
} # }
```
