# DatasetTemplate.delete

Deletes a specific dataset template on QuartzBio EDP.

## Usage

``` r
DatasetTemplate.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID or full name of a QuartzBio EDP dataset template.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetTemplate.delete(12345)
} # }
```
