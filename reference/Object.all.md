# Retrieves the metadata about all objects on EDP accessible to the current user.

Retrieves the metadata about all objects on EDP accessible to the
current user.

## Usage

``` r
Object.all(env = get_connection(), ...)
```

## Arguments

- env:

  Custom client environment.

- ...:

  Additional query parameters, passed to .request().

  <https://docs.solvebio.com/>
