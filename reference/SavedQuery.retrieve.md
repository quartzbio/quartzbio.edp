# SavedQuery.retrieve

Retrieves a specific saved query on QuartzBio EDP by ID.

## Usage

``` r
SavedQuery.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP saved query.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
SavedQuery.retrieve(12345)
} # }
```
