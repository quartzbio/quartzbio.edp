# Beacon.all

Retrieves the metadata about all beacons on QuartzBio EDP accessible to
the current user.

## Usage

``` r
Beacon.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters.

## Examples

``` r
if (FALSE) { # \dontrun{
Beacon.all()
} # }
```
