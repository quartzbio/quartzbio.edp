# Task.all

Retrieves the metadata about all tasks on QuartzBio EDP accessible to
the current user.

## Usage

``` r
Task.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters.

## Examples

``` r
if (FALSE) { # \dontrun{
Task.all()
} # }
```
