# SavedQuery.delete

Deletes a specific saved query on QuartzBio EDP.

## Usage

``` r
SavedQuery.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID of the QuartzBio EDP saved query.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
SavedQuery.delete(12345)
} # }
```
