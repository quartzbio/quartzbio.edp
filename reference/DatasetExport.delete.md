# DatasetExport.delete

Deletes a specific dataset export on QuartzBio EDP.

## Usage

``` r
DatasetExport.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset export.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetExport.delete(12345)
} # }
```
