# SavedQuery.create

Create a QuartzBio EDP saved query.

## Usage

``` r
SavedQuery.create(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Saved query attributes.

## Examples

``` r
if (FALSE) { # \dontrun{
SavedQuery.create(name = "My Dataset Template")
} # }
```
