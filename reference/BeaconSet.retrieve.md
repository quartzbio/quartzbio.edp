# BeaconSet.retrieve

Retrieves the metadata about a specific beacon set on QuartzBio EDP.

## Usage

``` r
BeaconSet.retrieve(id, env = get_connection())
```

## Arguments

- id:

  The ID of the beacon set.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
BeaconSet.retrieve("1234")
} # }
```
