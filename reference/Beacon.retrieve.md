# Beacon.retrieve

Retrieves the metadata about a specific beacon on QuartzBio EDP.

## Usage

``` r
Beacon.retrieve(id, env = get_connection())
```

## Arguments

- id:

  The ID of the beacon.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Beacon.retrieve("1234")
} # }
```
