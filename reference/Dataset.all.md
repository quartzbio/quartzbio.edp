# Retrieves the metadata about datasets on QuartzBio EDP.

Retrieves the metadata about datasets on QuartzBio EDP.

## Usage

``` r
Dataset.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters (e.g. page).

## Examples

``` r
if (FALSE) { # \dontrun{
Dataset.all()
} # }
```
