# Dataset.template

Retrieves the template for a dataset.

## Usage

``` r
Dataset.template(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Dataset.template("1234567890")
} # }
```
