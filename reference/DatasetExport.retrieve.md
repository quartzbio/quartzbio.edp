# DatasetExport.retrieve

Retrieves the metadata about a specific dataset export on QuartzBio EDP.

## Usage

``` r
DatasetExport.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset export.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetExport.retrieve(12345)
} # }
```
