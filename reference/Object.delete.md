# Object.delete

Delete a specific object from QuartzBio EDP.

## Usage

``` r
Object.delete(id, env = get_connection())
```

## Arguments

- id:

  The ID of the object.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Object.delete("1234567890")
} # }
```
