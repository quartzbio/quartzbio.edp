# DatasetTemplate.retrieve

Retrieves the metadata about a specific dataset template on QuartzBio
EDP.

## Usage

``` r
DatasetTemplate.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset template.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetTemplate.retrieve(12345)
} # }
```
