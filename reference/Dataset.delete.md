# Dataset.delete

Delete a specific dataset from QuartzBio EDP.

## Usage

``` r
Dataset.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Dataset.delete("1234567890")
} # }
```
