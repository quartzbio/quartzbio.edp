# DatasetTemplate.all

Retrieves the metadata about all dataset templates on QuartzBio EDP.

## Usage

``` r
DatasetTemplate.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters (e.g. page).

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetTemplate.all()
} # }
```
