# BeaconSet.all

Retrieves the metadata about all beacon sets on QuartzBio EDP accessible
to the current user.

## Usage

``` r
BeaconSet.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters.

## Examples

``` r
if (FALSE) { # \dontrun{
BeaconSet.all()
} # }
```
