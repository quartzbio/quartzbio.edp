# Task.retrieve

Retrieves the metadata about a specific task on QuartzBio EDP.

## Usage

``` r
Task.retrieve(id, env = get_connection())
```

## Arguments

- id:

  The ID of a task.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Task.retrieve("1234567890")
} # }
```
