# DatasetMigration.retrieve

Retrieves the metadata about a specific dataset migration on QuartzBio
EDP.

## Usage

``` r
DatasetMigration.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset migration.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetMigration.retrieve(12345)
} # }
```
