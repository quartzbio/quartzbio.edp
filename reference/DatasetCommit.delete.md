# DatasetCommit.delete

Deletes a specific dataset commit on QuartzBio EDP.

## Usage

``` r
DatasetCommit.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID or full name of a QuartzBio EDP dataset commit.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetCommit.delete(12345)
} # }
```
