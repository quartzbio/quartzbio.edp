# Vault.delete

Delete a specific vault from QuartzBio EDP. This operation cannot be
undone.

## Usage

``` r
Vault.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP vault.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Vault.delete("1")
} # }
```
