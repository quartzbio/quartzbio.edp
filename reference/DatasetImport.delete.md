# DatasetImport.delete

Deletes a specific dataset import on QuartzBio EDP.

## Usage

``` r
DatasetImport.delete(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP dataset import.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
DatasetImport.delete(12345)
} # }
```
