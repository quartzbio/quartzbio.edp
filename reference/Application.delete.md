# Application.delete

Delete a specific application from QuartzBio EDP.

## Usage

``` r
Application.delete(client_id, env = get_connection())
```

## Arguments

- client_id:

  The client ID for the application.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Application.delete("abcd1234")
} # }
```
