# Object.retrieve

Retrieves the metadata about a specific object on QuartzBio EDP.

## Usage

``` r
Object.retrieve(id, env = get_connection())
```

## Arguments

- id:

  The ID of the object.

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Object.retrieve("1234567890")
} # }
```
