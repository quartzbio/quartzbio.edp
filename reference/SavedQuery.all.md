# SavedQuery.all

Retrieves the all saved queries on QuartzBio EDP.

## Usage

``` r
SavedQuery.all(env = get_connection(), ...)
```

## Arguments

- env:

  (optional) Custom client environment.

- ...:

  (optional) Additional query parameters (e.g. page).

## Examples

``` r
if (FALSE) { # \dontrun{
SavedQuery.all()
} # }
```
