# Vault.retrieve

Retrieves the metadata about a specific QuartzBio EDP vault.

## Usage

``` r
Vault.retrieve(id, env = get_connection())
```

## Arguments

- id:

  String The ID of a QuartzBio EDP vault

- env:

  (optional) Custom client environment.

## Examples

``` r
if (FALSE) { # \dontrun{
Vault.retrieve("1234567890")
} # }
```
